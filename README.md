# E-Commerce Sales Analysis Using SQL

## Project Overview

This project analyzes retail sales data using SQL to identify revenue trends, regional performance, category-wise sales, profitability, and discount patterns. The analysis helps derive business insights that can support data-driven decision-making.

## Tools Used

* MySQL
* SQL
* GitHub

## Key Analysis Performed

* Total Sales and Profit Analysis
* Category-wise Revenue Analysis
* Region-wise Sales Analysis
* State-wise Sales Performance
* City-wise Revenue Analysis
* Profitability Analysis
* Discount Impact Analysis

## SQL Concepts Used

* SELECT
* GROUP BY
* ORDER BY
* Aggregate Functions (SUM, AVG)
* LIMIT
* Data Aggregation

## Key Insights

* Identified the highest revenue-generating categories.
* Analyzed regional sales performance.
* Evaluated profitability across categories.
* Determined top-performing states and cities.
* Examined discount patterns and their effect on sales.
